.. _getTime:

``getTime``
***********

This command returns the current time in the **Domain**.

.. function:: getTime

.. admonition:: Example:

   The following example is used to set the variable **currentTime** to current state of **time** in the **Domain**

   1. **Tcl Code**

   .. code-block:: tcl

	set currentTime [getTime]

   2. **Python Code**

   .. code-block:: python

	currentTime = model.getTime()
